# Claude Code Workflow - An Artist's AI Setup

How a filmmaker uses Claude Code as a daily creative and technical partner.

## Who

Ismaël Joffroy Chandoutis - filmmaker and artist based in Paris. Not a developer. Using Claude Code for vibe coding, context engineering, and building AI-powered creative tools.

## What's here

This repo documents the real workflow, decisions, and patterns that emerged from months of daily Claude Code usage. Not theory - actual config files, skills, and lessons learned.

### Setup
- `config/` - CLAUDE.md structure, soul.md philosophy, memory system
- `skills/` - Custom skills (interview-driven specs, creative pitch, etc.)
- `hooks/` - Automation hooks (notifications, backups)
- `mcp/` - MCP server configurations

### Journal
- `journal/` - Decisions log: what we tried, what worked, what we dropped
- `sources/` - Articles and threads that shaped our approach

### Patterns
- `patterns/` - Reusable patterns for specific use cases

## Philosophy

AI as collaborator that alters rather than augments. Not an assistant, not a productivity coach. A creative partner with memory and agency.
