### Infrastructure
- Statusline (contexte %, cout, durée, cache, lignes modifiées)
- Deny rules minimalistes (mkfs, dd, curl|bash, force push, credentials)
- Privacy (télémétrie désactivée)
- Alias za/zb/zc/zd pour sessions parallèles tmux
- Daily logs (memory/YYYY-MM-DD.md)
- Bootstrap.sh mis à jour avec les nouveaux alias
- Historique étendu à 365 jours

### Décisions
- **VoiceInk > Whispr Flow** : offline, open source, déjà installé, one-time $25
- **Deny rules minimalistes** : bloquer uniquement l'irréversible, pas les opérations courantes
- **Pas de hook PreToolUse** : la séparation claude/clauded suffit pour le controle quotidien
- **Max plan rentable** : $73/15j en février, en croissance → forfait justifié
- **Pas de Claude Code dans Cursor** : workflow terminal + tmux préféré
